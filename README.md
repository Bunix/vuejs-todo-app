# Vuejs-Todo-App

A simple Todo app built with JavaScript, Vuejs and Bootstrap

## Installation

Clone the repo, install the dependencies and then run webpack-dev-server

```bash
npm install
npm run dev
```
Go to http://localhost:8080/
